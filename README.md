# 📚 مُحلِّل الدرس بالأسئلة - Sultan

تطبيق تعليمي ذكي لتحليل النصوص الأدبية وتقديم بطاقات تحليلية شاملة، لوحة تعبيرية، وفيلم متحرك سينمائي.

## ✨ الميزات

- 🎯 **13 بطاقة تحليلية:** ماذا، كيف، لماذا، أين، متى، من، النتيجة، العاطفة، الخيال، الفكرة، البلاغة، الموسيقى، الخلاصة
- 🎨 **لوحة تعبيرية:** ألوان، إيموجي، شدة العاطفة، نبرة النص
- 🎬 **فيلم متحرك:** مشاهد سينمائية متتابعة تحاكي أحداث النص
- 📤 **إرسال PDF:** عبر واتساب، بريد، أو حفظ مباشر
- 📱 **PWA:** قابل للتثبيت على الجوال كتطبيق

## 🚀 خطوات النشر على Vercel

### الخطوة ١: احصل على مفتاح API

1. اذهب إلى [console.anthropic.com](https://console.anthropic.com)
2. أنشئ حساباً وسجّل دخول
3. من القائمة اختر **API Keys** ← **Create Key**
4. انسخ المفتاح (يبدأ بـ `sk-ant-...`) واحفظه مؤقتاً

### الخطوة ٢: ارفع المشروع

**الطريقة الأسهل (سحب وإفلات):**
1. أنشئ حساباً مجانياً على [vercel.com](https://vercel.com)
2. اضغط **Add New** ← **Project**
3. اختر **Import Third-Party Git Repository** أو اسحب ملف ZIP

**أو عبر GitHub:**
1. ارفع المجلد على GitHub
2. في Vercel: **Import Project** ← اختر الـ repo

### الخطوة ٣: أضف مفتاح API

في Vercel أثناء النشر:
1. اضغط على **Environment Variables**
2. أضف متغيراً جديداً:
   - **Name:** `ANTHROPIC_API_KEY`
   - **Value:** المفتاح الذي نسخته
3. اضغط **Add** ← **Deploy**

### الخطوة ٤: غيّر اسم الرابط

بعد النشر:
1. ادخل على المشروع في Vercel
2. **Settings** ← **Domains**
3. غيّر اسم الـ subdomain إلى `sultan` (أو ما تريد)
4. ✅ الرابط الجاهز: `https://sultan.vercel.app`

## 📱 تثبيت التطبيق على الجوال

افتح الرابط في:

- **Safari (آيفون):** زر المشاركة ← **إضافة إلى الشاشة الرئيسية**

## 🔧 التشغيل محلياً للتطوير

```bash
npm install
npm run dev
```

ضع المفتاح في ملف `.env.local`:
```
ANTHROPIC_API_KEY=sk-ant-...
```
